Upwork Portfolio Assistant

A Chrome extension that generates one portfolio draft per selected image, fills the Upwork portfolio modal, adds the five fixed deliverables, and lets you review before submission.

It supports a free Groq API key for AI-generated titles, roles, and descriptions, with a built-in fallback generator when no key is configured.

  1. Download or clone the GitHub project.
  2. Open chrome://extensions and enable Developer mode.
  3. Click Load unpacked and select the extension folder.
  4. Open extension settings and paste your Groq API key if you want AI generation.
